PaperPen

PaperPen គឺជា Software មួយដែលបង្កើតឡើងដើម្បីអោយអ្នកប្រើប្រាស់អាចរក្សាទុកទិន្នន័យ រឺ ព័ត៌មានរបស់ពួកគេបានយ៉ាងងាយស្រួលដោយការ Note ទុកនៅក្នុងកម្មវីធី ។ មិនត្រឹមតែប៉ុណ្ណោះ ពួកគេក៏អាច Add, Delete, Edit, Search នៅក្នុងនោះផងដែរ ព្រមទាំងអាចកំណត់ Note នោះជា Simple Note ក៏បាន និង Important Note ក៏បាន ។

Instuction

1. Home

Home page នៃ PaperPen System មានមុខងារដូចជា៖

  • – Button Search សម្រាប់ស្វែងរក Note ។

  • – សញ្ញា + តំណាងអោយការ Add Note ។

2. Add

ដំណើរការនៃការ Add Note ដំបូងយើងចុចលើសញ្ញា + នោះវានឹងបង្ហាញ Form មួយដែលអាចអោយយើងសរសេរអំពីអ្វីដែលយើងចង់ Note ទុក ដែលក្នុងនោះមានដូចជា៖

  • – Title : ជាចំណងជើងនៃ Note របស់យើង

  • – Content : ជាកម្រងអត្ថបទដែលតម្រូវអោយយើងសរសេររបាយការណ៍ រឺ សំណេរផ្សេងៗដែលយើងចង់ Note ទុក ។

  • – Date : ជាថ្ងៃខែ ពេលវេលា សម្រាប់អោយយើងកត់ចំណាំនៅលើ Note របស់យើង ។

  • – Important : ជាពាក្យមួយដែលអាចអោយយើងជ្រើសយក ប្រសិនបើយើងចង់បញ្ជាក់ថា Note របស់យើងជា Note Important ។ រាល់ Note ដែល important វានឹងលោតមកនៅខាងមុខ Simple Note ទាំងអស់ ទោះបីជា Important Note បានសរសេរក្រោយគេក៏ដោយ ។

Note:

  • – Important Note បញ្ជាក់ដោយពណ៌បៃតងចាស់

  • – Simple Note បញ្ជាក់ដោយពណ៌ផ្ទៃមេឃ

3. Edit

នៅពេលដែលយើងចង់ Edit Note របស់យើង ដោយហេតុផលចង់កែទិន្នន័យ រឺ ចង់សរសេរបន្ថែមទៀត នោះយើងគ្រាន់តែ Click on Note ហើយយើងអាចកែទិន្នន័យ រឺ សរសេរបន្ថែមបានហើយ ។ បន្ទាប់មក ចុចលើ Button Edit ដើម្បី Save ទិន្នន័យដែលបានកែប្រែរួច ។

4. Delete

ប្រសិនបើ យើងចង់ Delete Note របស់យើងណាមួយ នោះយើងគ្រាន់តែ Long Click on left Mouse អោយយូរ នៅលើ Note នោះ វានឹងបង្ហាញ Message alert មួយដែលសួរយើងបញ្ជាក់ថា ចង់លុប Note នេះដែររឺទេ?

  • – បើយើងចង់លុប នោះយើងត្រូវចុចលើ Button Yes

  • – បើយើងមិនចង់លុប នោះយើងត្រួវចុចលើ Button No ។

5. Search

នៅក្នុងករណីមួយដែលយើងមាន Note ច្រើនពេកដែលធ្វើអោយយើងពិបាកស្វែងរក ដំណោះស្រាយពេលនោះ ដោយយើងគ្រាន់តែចងចាំចំណងជើងរបស់ Note (Title) នោះយើងក៏នឺងអាច Search រក Note ដែលយើងចង់រកបានហើយ ព្រោះកម្មវិធីនឹង Hide Notes ទាំងអស់ដែលយើងមិនត្រូវការ ដែលបង្ហាញជំនួសវិញដោយ Note ដែលយើងបាន Search ។
