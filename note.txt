PaperPen is a Software build for saving data or information easily by noting in the app. You can also Add, Delete Edit, Search, etc. in the PaperPen. PaperPen lets you set kind of note as Simple or Important note.

Home page of PaperPen System has functions:

– Button Search for searching any notes.

– + used to presets for Adding Note.

– Date : ជាថ្ងៃខែ ពេលវេលា សម្រាប់អោយយើងកត់ចំណាំនៅលើ Note របស់យើង ។

ប្រសិនបើ យើងចង់ Delete Note របស់យើងណាមួយ នោះយើងគ្រាន់តែ Long Click on left Mouse អោយយូរ នៅលើ Note នោះ វានឹងបង្ហាញ Message alert មួយដែលសួរយើងបញ្ជាក់ថា ចង់លុប Note នេះដែររឺទេ?

– បើយើងចង់លុប នោះយើងត្រូវចុចលើ Button Yes

– បើយើងមិនចង់លុប នោះយើងត្រួវចុចលើ Button No ។

To delete any note, just long click with left mouse button on that note then it will alert the confirmation dialog asks you that you are sure to delete it.
– You really want to delete click on button Yes.

– You don't want to delete click on button No.
